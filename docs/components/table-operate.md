## 何时使用

- 需要一个带标题与下载区的表格容器，用于承载页面级的表格内容。
- 希望把“容器布局/通用操作区”与具体表格实现解耦，便于复用。
- 需要在不同页面以一致的结构包裹子组件，减少样式与布局重复。
