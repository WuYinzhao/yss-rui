## 何时使用

- 需要带统一样式的 `antd` Tabs 容器，用于承载同一页面内的多视图/多状态切换。
- 需要把切换逻辑与具体内容解耦，便于维护和扩展。
- 需要在表格/数据区域上保持风格一致的标签交互。
